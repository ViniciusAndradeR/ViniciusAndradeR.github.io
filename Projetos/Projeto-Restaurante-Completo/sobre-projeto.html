Sobre o Projeto

Projeto desenvolvido com o objetivo de colocar em prática o que aprendi durante meus últimos meses aprendendo HTML, CSS e JavaScript.

Este foi meu primeiro projeto completo desenvolvido (desconsiderando projetos simples como criação de páginas individuais para práticas). A ideia era criar, do zero, toda uma identidade e design de um site de um restaurante. Me inspirei em alguns modelos da plataforma dribbble.com, mais especificamente neste modelo do Muhsin Ahmed.

Todas as imagens usadas no site pertencem à plataforma unsplash.com, tive o cuidado de deixar um botão em cada imagem levando o usuário do site para os devidos créditos das imagens. Importante ressaltar que eu também tive o cuidado de carregar as imagens com uma dimensão bem menor que a original para que as páginas não fiquem muito pesadas para carregar.

Tanto as avaliações quanto os produtos do Cardápio foram incluídos através de objetos armazenados em variáveis de arquivos em JavaScript. Exemplo de um objeto:

{
id: 1,
titulo: "Cheese Torto",
categoria: "Lanche",
img: "https://unsplash.com/photos/6X...",
descricao: "Lorem ipsum dolor sit amet consectetur adipisicing elit. Quia odio, vero magnam cumque provident assumenda quis sit dolorum corrupti dignissimos.",
preco: 29.99,
destaque: true,
credito: "https://unsplash.com/photos/6XmQV6GccYU"
}

Dessa forma, é possível simular um site que gera conteúdo de forma "dinâmica", adicionando os objetos em elementos HTML através da função innerHTML. Pretendo reformular o site em breve para adicionar esses dados através de um framework como o NodeJS.
